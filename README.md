## Features

### 1. Spot the Bugs

Hunt down and fix subtle bugs hidden within the code. Challenge yourself with a variety of puzzles designed to sharpen your logical thinking.

### 2. Multi-Language Support

Practice in your go-to language or master a new one.

- **Supported:** TypeScript, JavaScript, Python, Vue, HTML/CSS.

### 3. Time Attack Mode

Compete with coders worldwide in a race against the clock. Efficiency is the key to the top score.

### 4. Developer Friendly

- **Works in your browser**: No local setup required for playing.
- **Modern UI**: Built with Vue 3 + Vuetify for a smooth, responsive experience.

## Getting Started

### Prerequisites

- Docker & Docker Compose

### Installation

1. **Clone the repository**

    ```bash
    git clone git@github.com:youtame/higashiyamanashi.git
    cd higashiyamanashi

    ```

1. build docker container

    ```bash
    docker compose up -d --build
    ```

1. visit your site
   [http://localhost:5173](http://localhost:5173 "http://localhost:5173")
